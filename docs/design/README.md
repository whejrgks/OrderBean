# 디자인 문서

## UI/UX 디자인

### 주요 화면
- 홈 화면
- 지도 검색 화면
- 카페 상세 화면
- 취향 설문 화면
- 관리자 대시보드

### 디자인 원칙
- 지도 중심의 직관적 탐색
- 취향 일치도 시각화
- 모바일 친화적 디자인
- 접근성 고려 (WCAG 준수)

## 데이터 모델

### 사용자 (User)
- 기본 정보 (이메일, 이름)
- 취향 프로필 (산미, 로스팅, 바디감 등)

### 카페 (Cafe)
- 위치 정보 (위도, 경도)
- 기본 정보 (이름, 주소, 전화번호)
- 평균 맛 데이터 (리뷰 기반)

### 리뷰 (Review)
- 리뷰 내용
- 맛 태그 및 점수
- 평점
